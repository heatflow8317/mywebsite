Log-Harnack inequality

  • Suppose \(u: B_2(0)\rightarrow \mathbb{R}\) is positive harmonic, then \[\sup_{B_1(0)}\frac{|\nabla u|}{u} \leq c\] for some dimensional constant \(c\).

  • Proof:

    Define \(v=\log u\) then \[\nabla v = \nabla \log u = \frac{\nabla u}{u}\] \[\Delta v = \nabla \frac{\nabla u}{u} = \frac{u\Delta u-|\nabla u|^2}{u^2}=-\frac{|\nabla u|^2}{u^2}\] Define \(w=|\nabla v|^2\) such that \(\Delta v = -w\). Then \[\nabla w = \nabla |\nabla v|^2 = (v^2_i)_j=2v_{ij}v_i\] \[\Delta w = (2v_{ij}v_i)_j = 2v_{ijj}v_i + 2v^2_{ij} = -2w_iv_i + 2v^2_{ij}\] Let \(\varphi\in C^\infty_c (B_2)\) such that \(1\geq \varphi \geq 0\) and \(\varphi=1\) on \(B_1\). Then \[\Delta (\varphi w) = \varphi \Delta w + w\Delta \varphi + 2\nabla \varphi \nabla w= -2\varphi w_i v_i + 2\varphi v^2_{ij} + w\Delta \varphi + 4\nabla \varphi v_{ij}v_i\] \[\Delta (\varphi w) + 2\varphi w_iv_i + 2\varphi_i wv_i = 2\varphi v^2_{ij} + w\Delta \varphi + 4\nabla \varphi v_{ij} v_i + 2\varphi_i wv_i\] \[\Delta (\varphi w) + 2\nabla v \nabla (\varphi w) = 2\varphi v^2_{ij} + w\Delta \varphi + 4\varphi_j v_{ij} varphi_i + 2\varphi_i wv_i\] \[=\varphi v^2_{ij} + w\Delta \varphi + 2w\varphi_iv_i + (\varphi v^2_{ij} + 4\varphi_j v_i v_{ij})\] \[\geq \varphi v^2_{ij} + w\Delta \varphi + 2w\varphi_iv_i +(\varphi v^2_{ij} + 4\varphi^{\frac{1}{2}}\varphi_j v_i v_{ij})\] \[\geq \varphi v^2_{ij} + w\Delta \varphi + 2w\varphi_iv_i -4v^2_i\varphi$2_j \] \[\geq \varphi v^2_{ij} + c_1 w - c_2 |\nabla v|w - 4c_3 |\nabla v|^2\] \[\geq \varphi \frac{w^2}{n} + c_1 w - c_2 w^{\frac{3}{2}}+c_3w\] \[\geq P_4(\varphi^{\frac{1}{4}}w^{\frac{1}{2}}\] where \(P_4\) is a polynomial of degree 4 and leading coefficient \(\frac{1}{n}>0\).
  • Let \(x\) be a point where \(\varphi w\) reaches interior maximum, then \[0\geq P_4\] Polynomial like \(P_4\) must be positive when \(|\varphi^{\frac{1}{2}}w|\) is big, so this implies \[|\varphi^{\frac{1}{2}}w|\leq k\] for some dimensional \(k>0\). On \(B_1\), this means \[\frac{|\nabla u|}{u}\leq k\]
