Mean Value Property and Harmonic function

First of all, we list the two mean value properties: let \(u\in C(\Omega\subset \mathbb{R}^n)\) of connected domain, and \(\omega(n)=|\mathbb{S}^{n-1}\subset \mathbb{R}^n|\). We have \(n\alpha(n)=\omega(n)\) where \(\alpha(n)\) is volume for \(n\)-dimensional solid ball.
  • (Area mean) \[u(x)=\frac{1}{\omega(n)r^{n-1}}\int_{\partial B_r(x)} u(y)\mathrm{d}S_y\]
  • (Volume mean) \[u(x)=\frac{n}{\omega_nr^n}\int_{B_r(x)}u(y)\mathrm{d}y\]

  • Maximum principle:

    Suppose \(u\in C(\overline{\Omega})\) satisfies the mean value property in \(\Omega\), then \(u\) attains maximum on the boundary, or \(u\) is constant.
  • Proof: We claim that the set \[\Sigma=\{x\in \Omega: u(x)=M=\max_{\overline{\Omega}}u\}\subset \Omega.\] As a level set it is relatively closed. For openness, pick any \(z\in \Sigma\) and \(B_r(z)\subset \Omega\), mean value property implies \(u=M\) on the whole ball \(B_r(z)\).

  • Harmonic implies MVP:

  • Proof: By change of variables, deduce that \[\partial_r \int_{|w=1|}u(x+rw)\mathrm{d} S_w=0\] Integrate from 0 to \(r\).

  • MVP implies smooth and harmonic:

  • Proof: For smoothness, let \(\epsilon < d(x,\partial \Omega)\), check that \(\varphi_\epsilon * u(x)=u(x)\) for all \(x\in \Omega\), where \(\varphi_\epsilon\) is mollifier and \(\varphi\in C^\infty_0(B_1(0))\) is radial and \(\int_{B_1(0)}\varphi=1\). For harmonicity, apply mean value property to \[\int_{B_r(x)}\Delta u = r^{n-1}\partial_r \int_{|w|=1}u(x+rw)\mathrm{d}S_w.\]

  • Gradient estimate:

    Suppose \(u\in C(\overline{B}_R)\) is harmonic in \(B_R=B_R(x_0)\). Then we have \[|Du(x_0)|\leq \frac{n}{R}\max_{\overline{B}_R}|u|.\]
  • Proof: we note that Green's formula can be used in this way: \[\int_\Omega u_i = \int_\Omega e_i \cdot \nabla u = \int_\Omega u \Delta x_i + \nabla x_i \cdot \nabla u = \int_{\partial \Omega} u \partial_\nu x_i=\int_{\partial \Omega} u \nu_i.\] where \(\Delta x_i=0\).
  • Remark: a harmonic function in \(\mathbb{R}^n\) bounded from above or below is constant.

  • Higher gradient estimate:

    Suppose \(u \in C(\overline{B}_R)\) is harmonic in \(B_R=B_R(x_0)\), Then for any multi-index \(\alpha\) with \(|\alpha|=m\) we have \[|D^\alpha u(x_0)|\leq \frac{n^m e^{m-1} m!}{R^m} \max_{\overline{B}_R} |u|\]

  • Harmonic function is analytic.

  • Harnack inequality:

    Suppose \(u\) is harmonic in \(\Omega\). Then for any compact subset \(K\subset \Omega\) there exists a constant \(C=C(\Omega, K)>0\) such that if \(u\geq 0\) in \(\Omega\), then \[\frac{1}{C}u(y)\leq u(x) \leq Cu(y)\] for any \(x, y\in K\).
